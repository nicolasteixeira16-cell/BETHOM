# BETHOM

## Description

BETHOM is Thomas's personal portfolio, a graphic designer passionate about code and design. This website showcases his work, blog articles, and allows contacting him for UI design, illustration, and typography projects.

The site is entirely static, built with HTML and CSS, featuring a responsive and modern layout.

## Features

- **Home**: Introduction to Thomas with a call to action to download his CV.
- **Blog**: Recent articles on design systems and creating pixel-perfect icons in Figma.
- **Portfolio**: Gallery of featured works, including dashboards, illustrations, and typography projects.
- **Contact**: Contact form and contact information.

## Technologies Used

- **HTML5**: Site structure
- **CSS3**: Layout and styles
- **Google Fonts**: Heebo font for modern typography
- **Images**: Graphic assets stored in the `image/` folder

## Installation and Usage

1. Clone or download the project to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. The site will display directly without requiring a server or additional installation.

## Responsive Design

The site is designed to be responsive and adapt to different screen sizes:
- Desktop: Full layout with horizontal navigation
- Mobile: Adaptive navigation with burger menu (CSS implementation)
